# Shared Agent Prompts

Detta bibliotek innehaller delade promptfiler for AI-agenter som arbetar med dbt, Databricks och Customer 360-projektet.

## Syfte

Prompterna ska gora det enkelt for hela teamet att anvanda samma arbetsflode for:

- source onboarding
- modellering
- testning
- dokumentation
- review
